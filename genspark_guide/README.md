## 젠스파크란?

### 한 줄 정의

**"PPT 만드는 데 3시간? 젠스파크는 3분이면 됩니다"**

젠스파크는 GPT, Claude, Gemini 등 여러 AI 모델을 통합한 올인원 생산성 플랫폼입니다.
검색, 슬라이드 제작, 데이터 분석, 이미지/영상 생성 등 거의 모든 콘텐츠 업무를 한 곳에서 처리합니다.

### 창립 배경

전직 바이두 임원들이 2024년 6월에 창립한 AI 에이전트 플랫폼으로, 단순한 도구가 아닌 **자율적으로 복잡한 작업을 처리하는 AI 워크스페이스**를 지향합니다.

### 실제 효과

| 작업 유형 | 기존 소요 시간 | 젠스파크 | 절감율 |
|---------|--------------|---------|--------|
| 마케팅 보고서 작성 | 6시간 | 45분 | **85%** |
| PPT 제작 | 3시간 | 3분 | **98%** |
| 데이터 분석 보고서 | 8-10시간 | 15분 | **97%** |

**평균 업무 시간 절감: 85-90%**

### 젠스파크의 차별점

- **다중 AI 모델 통합**: GPT, Claude, Gemini를 한 번에 사용
- **완전 자동화**: 검색부터 최종 결과물까지 AI가 처리
- **실시간 웹 검색**: 최신 정보 기반 작업
- **올인원 워크스페이스**: 여러 도구를 오갈 필요 없음
- **MoA 시스템**: 여러 AI가 협업하여 최적의 결과 도출

## 핵심 기능 상세

### 1. AI 검색 (Sparkpage)

**개요**
기존 검색 엔진과 달리 실시간 웹 검색 후 AI가 정보를 요약하고 구조화하여 제공합니다.

**특징**
- 최신 정보 실시간 수집
- 여러 출처를 종합한 요약 제공
- 참조 링크 자동 첨부
- 후속 질문 추천

**사용 예시**
```
질문: "2025년 AI 트렌드는?"
→ 실시간 웹 검색 → 최신 기사/보고서 종합 → 구조화된 요약 제공
```

**크레딧**: 18 크레딧/검색

**활용 시나리오**
- 시장 조사 및 경쟁사 분석
- 최신 기술 트렌드 파악
- 뉴스 및 업계 동향 모니터링

### 2. AI 슬라이드

**개요**
데이터나 텍스트를 업로드하면 전문가 수준의 프레젠테이션을 자동으로 생성합니다.

**지원 형식**
- Excel (.xlsx, .xls)
- CSV
- PDF 문서
- 텍스트 입력
- 다중 문서 통합

**자동 생성 요소**
- 표지 (제목, 부제목, 날짜)
- 목차 (섹션 자동 구조화)
- 섹션 제목 페이지
- 글머리 기호 페이지 (요약)
- 표 형식 페이지 (상세 데이터)
- 차트/그래프 (자동 시각화)
- 다이어그램 (프로세스, 구조)
- 타임라인/일정표
- 강조 페이지 (핵심 메시지)
- Q&A/마무리 페이지

**템플릿 시스템**
- 템플릿 ID 저장 및 재사용
- 브랜딩 일관성 유지
- 색상/폰트/레이아웃 자동 적용

**크레딧**: 145 크레딧/슬라이드

**내보내기 형식**: PPTX, PDF

**활용 사례**
- 인증 준비 가이드
- 실적 보고서
- 영업 제안서
- 교육 자료
- 연구 발표 자료

### 3. AI 시트

**개요**
데이터 수집, 정리, 분석, 시각화를 자동으로 처리하는 스프레드시트 자동화 도구입니다.

**주요 기능**
- **자동 데이터 크롤링**: 웹에서 데이터 수집
- **데이터 정리**: 중복 제거, 형식 통일
- **자동 분석**: 통계, 트렌드 분석
- **시각화**: 차트/그래프 자동 생성
- **대시보드 구성**: 실시간 데이터 모니터링

**사용 예시**

**마케팅 캠페인 성과 분석**
```
입력: 광고 플랫폼 데이터
→ ROI 계산, 채널별 성과 비교, 최적화 제안
```

**스타트업 정보 수집**
```
입력: "Y Combinator 2025 배치 스타트업 리스트"
→ 자동 크롤링 → 분류/정리 → 인사이트 도출
```

**소셜미디어 대시보드**
```
입력: SNS 계정 연동
→ 실시간 지표 수집 → 대시보드 생성
```

**크레딧**: 166 크레딧/시트

**활용 시나리오**
- 리드 생성 및 관리
- 경쟁사 모니터링
- 시장 데이터 수집
- 성과 지표 추적

### 4. AI 이미지 스튜디오

**개요**
텍스트 프롬프트로 이미지를 생성하고 편집하는 도구입니다.

**주요 기능**
- **이미지 생성**: 텍스트 → 이미지
- **이미지 편집**: 배경 제거, 객체 추가/삭제
- **스타일 변환**: 사진 → 일러스트, 스타일 변경
- **이미지 확장**: 아웃페인팅
- **업스케일링**: 해상도 향상

**생성 가능 이미지 유형**
- 제품 사진
- 마케팅 비주얼
- 소셜미디어 썸네일
- 일러스트레이션
- 인포그래픽 요소
- 로고 및 아이콘

**활용 사례**
- 블로그 썸네일 제작
- 광고 크리에이티브
- 프레젠테이션 삽화
- SNS 콘텐츠 제작

### 5. AI 비디오

**개요**
텍스트나 스크립트로 영상을 자동 생성하는 도구입니다.

**주요 기능**
- **스크립트 → 영상**: 텍스트 입력만으로 영상 제작
- **음성 합성**: AI 보이스오버 자동 생성
- **자동 편집**: 장면 전환, 효과 추가
- **배경 음악**: 자동 배경음 삽입
- **자막 생성**: 자동 자막 생성

**생성 가능 영상 유형**
- 설명 영상 (Explainer Video)
- 제품 소개 영상
- 교육 영상
- 소셜미디어 숏폼
- 광고 영상

**워크플로우 예시**
```
레시피 영상 제작:
1. 레시피 조사 (AI 검색)
2. 스크립트 작성 (AI 채팅)
3. 음성 합성 (AI 비디오)
4. 영상 편집 (자동)
5. 최종 영상 생성
```

**활용 사례**
- YouTube 콘텐츠
- 제품 데모 영상
- 온라인 강의
- 마케팅 캠페인

### 6. 슈퍼 에이전트

**개요**
복잡한 다단계 작업을 자율적으로 처리하는 AI 에이전트입니다.

**특징**
- 자율적 작업 수행
- 여러 도구를 연계하여 사용
- 실시간 의사결정
- 단계별 진행 상황 보고

**가능한 작업**

**여행 계획 수립**
```
입력: "파리 5일 여행 계획"
→ 항공편 검색 → 숙소 추천 → 일정 구성 → 예산 계산
→ 맛집 예약 정보 → 완전한 여행 가이드 생성
```

**마케팅 아웃리치 캠페인**
```
입력: "SaaS 제품 런칭 캠페인"
→ 타겟 고객 분석 → 메시지 작성 → 채널 선정
→ 일정 계획 → 추적 지표 설정 → 실행 계획서 생성
```

**영상 제작 프로젝트**
```
입력: "요리 레시피 영상 제작"
→ 레시피 조사 → 스크립트 작성 → 음성 생성
→ 영상 편집 → 썸네일 제작 → 완성
```

**활용 사례**
- 복잡한 프로젝트 기획
- 다단계 리서치 작업
- 자동화된 워크플로우 실행
- 통합 콘텐츠 제작

### 7. AI 채팅 (MoA)

**개요**
MoA(Mixture of Agents) 시스템으로 여러 AI 모델을 동시에 활용하는 채팅 인터페이스입니다.

**지원 모델**
- OpenAI GPT-4, GPT-4 Turbo
- Anthropic Claude 3 (Opus, Sonnet, Haiku)
- Google Gemini Pro
- 기타 오픈소스 모델

**MoA 시스템 작동 방식**
```
질문 입력
  ↓
여러 AI 모델이 동시에 답변 생성
  ↓
각 답변을 비교/분석
  ↓
최적의 답변 선택 또는 통합
  ↓
최종 답변 제공
```

**특징**
- 모델별 답변 비교
- 최고의 답변 자동 선택
- 다양한 관점 확인 가능
- 복잡한 문제 해결에 유리

**활용 사례**
- 중요한 의사결정
- 복잡한 문제 분석
- 창의적 아이디어 발굴
- 기술적 검증

### 8. AI 드라이브

**주요 기능**
- 작업물 자동 저장
- 폴더 구조 관리
- 버전 관리

- 검색 기능

**저장 가능 파일**
- AI 슬라이드 (PPTX, PDF)
- AI 시트 (XLSX, CSV)
- AI 이미지 (PNG, JPG)
- AI 비디오 (MP4)
- 프로젝트 파일

**저장 용량**
- 무료: 1GB
- Plus: 50GB
- Pro: 1TB

## 통합 워크플로우 예시

젠스파크의 진정한 힘은 여러 기능을 연계하여 사용할 때 발휘됩니다.

### 사례 1: 마케팅 캠페인 전체 프로세스

```
1. AI 검색 (Sparkpage)
   "2025년 B2B SaaS 마케팅 트렌드"
   → 최신 트렌드 조사 및 요약
```

- 통일된 형식 (날짜, 숫자, 텍스트)
- 불필요한 빈 공간 제거

### 2. 프롬프트 너무 모호

**❌ 나쁜 예:**
```
"프레젠테이션 만들어줘"
```

**✅ 좋은 예:**
```
템플릿 ID example-template-id...를 활용하여 Q1 실적 보고서 생성.
포함:
- 매출 차트 (막대)
- 전년 대비 증감률 표

- 다음 분기 목표 (타임라인)
```

### 3. 생성 결과 그대로 사용

**문제:**
- AI가 데이터 해석 오류 가능

- 비즈니스 맥락 누락 가능

**해결:**
- 반드시 프리뷰 확인
- 핵심 메시지 재검토
- 파워포인트에서 최종 편집

### 4. 템플릿 ID 미관리

**문제:**
- 템플릿 ID 분실
- 일관성 없는 디자인
- 재작업 필요

**해결:**
- 노션/문서에 템플릿 ID 기록
- 프로젝트별 템플릿 매핑
- 버전 관리

### 5. 참고 이미지 미활용

**놓치는 기회:**
- 특정 스타일 재현 불가
- 디자인 요구사항 전달 어려움

**활용 방법:**
- 원하는 디자인 스크린샷 첨부
- "이 이미지처럼..." 프롬프트
- 레이아웃 참고 이미지 제공

## 핵심 원칙

### 젠스파크 활용 4대 원칙

**1. 데이터가 핵심**
- 구조화된 데이터 = 완성도 높은 결과물
- 쓰레기 입력 = 쓰레기 출력 (Garbage In, Garbage Out)
- 데이터 품질이 AI 출력 품질을 결정

**2. 기능 연계가 힘**
- 단일 기능 < 여러 기능 연계
- AI 검색 → AI 시트 → AI 슬라이드 = 완전한 워크플로우
- 슈퍼 에이전트로 전체 프로세스 자동화

**3. 템플릿이 자산**
- 한 번 만든 템플릿 = 무한 재사용
- 브랜딩 일관성 유지
- 팀 생산성의 핵심

**4. 검토는 필수**
- AI 생성 후 반드시 검토
- 데이터 정확성 확인
- 비즈니스 맥락 재점검
